# Trier Digital Twin 🏛️💻

A comprehensive 3D Digital Twin of Trier, Germany. This project bridges the gap between historical preservation and modern urban data science.

## 🚀 Key Features
* **Real-time Integration:** Live weather and traffic status via OpenWeatherMap and traffic APIs.
* **Geospatial Visualization:** 3D rendering of Trier's terrain and modern housing density using CesiumJS.
* **Historical Context:** Mapping and visualization of Roman structures (Porta Nigra, Amphitheatre).
* **Robust Backend:** Secure Flask API with rate-limiting and PostgreSQL integration.

## 🛠️ Tech Stack
* **Frontend:** JavaScript, CesiumJS, CSS (Gold/Dark Theme)
* **Backend:** Python (Flask), PostgreSQL (PostGIS)
* **Tools:** Git LFS (for large GeoJSON datasets), Windsurf
